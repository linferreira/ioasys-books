# ioasys Books 

## Tecnologias utilizadas
- TypeScript
- Next.JS
- Styled Components
- React Query 
- MUI

### Passos para rodar o projeto
* Utilize o comando abaixo para instalar as dependências necessárias  
```
npm install
```

* Rode o projeto utilizando o comando

```
npm run dev
```
